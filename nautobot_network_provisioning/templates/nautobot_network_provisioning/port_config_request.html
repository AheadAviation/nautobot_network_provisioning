How it works:
  1. Select the Building where the port is located
  2. Enter the Communications Room identifier
  3. Enter the Jack identifier
  4. Choose the Service to apply
  5. Set the Schedule Time for the change
  6. Click Submit Request

The system will automatically find the switch and port, select the appropriate template, and schedule the configuration change.
