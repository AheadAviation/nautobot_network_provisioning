# Security Policy

## Supported Versions

Only the latest version of this project is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

We take security seriously. If you find a security vulnerability in this project, please report it to us by following these steps:

1.  **Do not** open a public issue on GitHub.
2.  Send an email to [INSERT EMAIL ADDRESS] with a detailed description of the vulnerability.
3.  Include steps to reproduce the vulnerability, along with any relevant logs or screenshots.
4.  Wait for a response from the project maintainers. We will acknowledge your report and provide an estimated timeline for a fix.

We will work with you to ensure that the vulnerability is addressed promptly and that users are notified when a fix is available.

Thank you for helping to keep this project secure!
